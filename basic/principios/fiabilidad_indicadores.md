# Sistema de Fiabilidad de Indicadores

Cada indicador tiene un nivel de fiabilidad del 1 al 3:

- **⭐⭐⭐ (3):** Indicador fuerte — alta fiabilidad, base sólida para tomar decisión
- **⭐⭐ (2):** Indicador medio — útil como confirmación, no como base única
- **⭐ (1):** Indicador débil — informativo pero poco fiable, puede engañar

## Tabla de Fiabilidad

| Indicador | Fiabilidad | Notas |
|-----------|------------|-------|
| Soporte/resistencia con múltiples toques | ⭐⭐⭐ (3) | Niveles probados históricamente, base sólida |
| Curva de futuros (contango/backwardation) | ⭐⭐⭐ (3) | Muestra dónde pone el dinero el mercado real |
| Fear & Greed Index | ⭐⭐ (2) | Útil para contexto, no para timing exacto |
| Noticias de medios | ⭐ (1) | Ruido, narrativa, posible manipulación |
| Pre-market | ⭐ (1) | Bajo volumen, fácilmente manipulable |
